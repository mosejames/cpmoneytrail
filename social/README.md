# CP Money Trail — Social Media Graphics

15 standalone HTML cards designed for screenshotting and posting to Facebook/Instagram. Each renders at 1200x630px (standard social card dimensions).

## How to Screenshot

1. Open any card HTML file in Chrome
2. The card renders at exactly 1200x630px
3. To capture:
   - **Chrome DevTools method:** Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows), type "Capture screenshot", select "Capture full size screenshot"
   - **Manual method:** Set device toolbar to 1200x630, then capture screenshot
   - **Quick method:** Right-click > Inspect > Toggle device toolbar > Set to 1200x630 > Cmd+Shift+P > "Capture screenshot"
